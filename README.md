# 车贷房贷计算器微信小程序

原生微信小程序，无第三方依赖。当前版本覆盖：

- 等额本息、等额本金、先息后本月供测算
- 贷款类型支持车贷 / 房贷
- 车贷快捷期数：12/24/36/48/60 期；房贷快捷期数：5/10/20/30 年
- 车贷工具：算月供、查真利率、平息换算
- 房贷工具：算月供、组合贷、能贷多少、提前还款
- 利率输入支持年化%、月息%
- 根据本金、期数、实际月供反推真实月利率和年化
- 银行“月息/平息”口径换算真实 APR
- 等额本息提前还款估算

## 开发

1. 用微信开发者工具打开本目录。
2. 当前 `project.config.json` 已写入 AppID；预览/上传需要当前微信号是该小程序开发者。
3. 本地计算核心自检：

```bash
npm test
```

## 计算口径

- 年化%输入按名义年利率处理，月利率 = 年化利率 / 12。
- 月息%输入会自动折算为年化利率参与还款计算。
- “真实利率”按现金流 IRR 反推。
- “平息/月息”按本金全期计息，再用实际月供反推真实资金成本。
