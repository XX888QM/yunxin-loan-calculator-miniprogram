# 车贷房贷计算器微信小程序开发文档

## 1. 产品定位

做一个面向普通用户和车商/房产销售的贷款计算工具，核心不是只算月供，而是把银行、金融公司常见的“月息”“平息”“手续费”“等额本息”口径拆开，帮用户看清真实资金成本。

一句话：让用户输入本金、期数、月供或利率后，马上知道月供、总利息、还款明细、真实月利率和真实年化。

## 2. 目标用户

- 买车贷款用户：想知道销售口中的月息到底贵不贵。
- 房贷用户：想快速计算商贷、公积金、组合贷和提前还款。
- 车商业务人员：给客户展示不同首付、期数、利率下的真实成本。
- 云鑫租车后续平台：可作为客户报价、合同测算和真实利率说明工具。

## 3. 核心功能

### 3.0 贷款类型

贷款类型：

- 车贷
- 房贷

选择 `组合贷` 工具时，系统默认切到 `房贷`。

快捷期数：

- 车贷：12/24/36/48/60 期
- 房贷：5/10/20/30 年

工具显示：

- 车贷：算月供、查真利率、平息换算
- 房贷：算月供、组合贷、能贷多少、提前还款

### 3.1 月供测算

输入：

- 贷款类型：车贷 / 房贷
- 贷款本金
- 还款期数
- 利率
- 利率口径：年化% / 月息%
- 还款方式

支持方式：

- 等额本息
- 等额本金
- 先息后本

输出：

- 首月/月供
- 末期还款
- 总利息
- 还款总额
- 完整还款明细

### 3.2 房贷组合贷

输入：

- 贷款类型：房贷
- 商贷本金
- 商贷利率
- 公积金本金
- 公积金利率
- 利率口径：年化% / 月息%
- 贷款年限
- 还款方式

输出：

- 合计月供
- 总利息
- 还款总额
- 每期本金、利息、余额

### 3.3 月供预算反推

输入：

- 月供预算
- 还款期数
- 利率
- 利率口径：年化% / 月息%
- 还款方式：等额本息 / 等额本金

输出：

- 最高可贷本金
- 首月/月供
- 总利息
- 还款总额
- 完整还款明细

用于回答“我每月最多还多少，能贷多少钱”。

### 3.4 真实利率反推

用于处理银行或金融公司已经给出还款计划的情况。

输入：

- 贷款本金
- 实际月供
- 还款期数
- 对外宣称年化% 或月息%

输出：

- 真实月利率
- 名义年化
- 复利年化
- 总利息
- 真实月息与宣称月息的差值
- 真实成本是宣称口径的多少倍

典型场景：银行说月息 0.59%，但还款表按 1.095% 月利率计算。

### 3.5 平息/月息换算

用于解释“月息”和“平息”不是同一个真实资金成本。

输入：

- 贷款本金
- 平息费率
- 平息口径：年化% / 月息%
- 还款期数

输出：

- 平息口径月供
- 总利息
- 反推后的真实月利率
- 真实年化

关键说明：平息是按原始本金全期计息，真实资金占用会逐月减少，所以平息 0.59% 通常不等于真实月利率 0.59%。

### 3.6 提前还款估算

输入：

- 贷款本金
- 利率
- 利率口径：年化% / 月息%
- 总期数
- 已还期数
- 提前还款金额
- 处理方式：缩短期限 / 降低月供

输出：

- 当前剩余本金
- 提前还款后本金
- 预计节省利息
- 新剩余期数
- 新月供

## 5. 页面结构

采用一个首页多工具切换，不先做多页面。

原因：

- 小程序体量小，打开即用。
- 用户无需学习导航。
- 计算结果可以快速复制。
- 后续要拆页面也容易。

页面工具按贷款类型动态显示：

- 车贷：算月供、查真利率、平息换算
- 房贷：算月供、组合贷、能贷多少、提前还款

每个 Tab 结构：

- 贷款类型选择
- 输入区
- 结果卡片
- 明细表
- 复制结果按钮

## 6. 计算口径

### 6.1 等额本息

月利率：

```text
r = 年化利率 / 12
```

月供：

```text
月供 = 本金 * r * (1 + r)^期数 / ((1 + r)^期数 - 1)
```

每期：

```text
当期利息 = 剩余本金 * r
当期本金 = 月供 - 当期利息
剩余本金 = 上期剩余本金 - 当期本金
```

### 6.2 等额本金

每期本金固定：

```text
每期本金 = 本金 / 期数
```

每期：

```text
当期利息 = 剩余本金 * 月利率
当期还款 = 每期本金 + 当期利息
```

### 6.3 先息后本

每月先还利息，最后一期归还全部本金。

```text
每月利息 = 本金 * 月利率
最后一期还款 = 本金 + 每月利息
```

### 6.4 真实利率反推

通过现金流 IRR 反推。

已知：

- 本金
- 每月还款
- 期数

求一个月利率 r，让未来每期还款折现后等于本金：

```text
本金 = Σ 每月还款 / (1 + r)^期数
```

当前实现用二分法求 r，不依赖第三方库。

### 6.5 平息换算

平息总利息：

```text
总利息 = 本金 * 平息月费率 * 期数
```

平息月供：

```text
月供 = (本金 + 总利息) / 期数
```

再把这个月供放入 IRR 反推，得到真实月利率和真实年化。

## 7. 技术方案

技术栈：

- 原生微信小程序
- WXML + WXSS + JavaScript
- 无第三方依赖
- 计算逻辑集中在 `utils/loan.js`

目录：

```text
app.js
app.json
app.wxss
project.config.json
sitemap.json
pages/index/index.js
pages/index/index.wxml
pages/index/index.wxss
utils/loan.js
tests/loan.test.js
```

设计原则：

- 公式只写在 `utils/loan.js`
- 页面只负责输入、展示、复制
- 所有金额保留 2 位小数
- 年化%输入使用百分比，例如 `13.14` 表示 13.14%。
- 月息%输入使用百分比，例如 `0.59` 表示月息0.59%。
- 普通利率选择月息%时按 `月息 * 12` 折算为年化。
- 平息选择年化%时按 `年费率 / 12` 折算为月费率。
- 真实月利率展示到 4 位百分比，方便看出细微差异

## 8. UI 方向

风格：金融账本感，克制、清楚、可信。

视觉要求：

- 不做花哨营销页，首页直接是工具。
- 重点数字放大显示。
- 输入项密集但不拥挤。
- 结果卡片用深绿色强调真实成本。
- 明细表按输入期数展示完整明细，金额列支持横向滚动。


```text
商贷：1000000，年化 3.45%
```

```text
能输出合计月供、总利息、明细表

```

## 11. 合规和文案边界

页面文案不要直接写“银行骗人”。

建议用：

- 真实资金成本
- 宣称口径
- 平息口径
- 反推年化
- 仅供测算，具体以合同和银行还款计划为准

- 车贷 3 个工具、房贷 4 个工具都能计算。

- 能复制当前结果。
- `npm test` 通过。
- 平安银行 35 万车贷样例能反推出约 1.095% 月利率。
